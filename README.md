# スクリーンカーテン

ページをオーバーレイで覆い隠す Chrome 拡張機能です。
見たくない情報を隠したり、離席時のプライバシー保護に活用できます。

## 機能

- **3つのテーマ** — Light / Dark / Glass（ぼかし強度調整可能）
- **ドラッグ移動** — オーバーレイを自由に移動
- **8方向リサイズ** — 角・辺のハンドルでサイズ調整
- **設定の自動記憶** — 位置・サイズ・テーマを自動保存、次回復元

## 使い方

1. ツールバーのアイコンをクリック
2. テーマを選択（Glass の場合はぼかし強度も調整可能）
3. 「オーバーレイを表示」ボタンを押す

オーバーレイを閉じるには右上の × ボタンをクリックしてください。

## 開発

```bash
npm install
npm run build   # アイコン + ストア用スクリーンショットを生成
```

Chrome で `chrome://extensions` を開き、「パッケージ化されていない拡張機能を読み込む」からプロジェクトフォルダを選択して動作確認できます。

## 権限

| 権限 | 用途 |
|------|------|
| `activeTab` | 現在のタブにオーバーレイを表示 |
| `scripting` | コンテンツスクリプトと CSS の動的注入 |
| `storage` | 設定のローカル保存・復元 |

外部サーバーとの通信は一切行いません。
